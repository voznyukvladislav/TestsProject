Main purpose of this software is providing users with testing functional.

Test is a collection of different questions, which can be related to one or many categories. Points for each type is specified in brackets. Each question belongs to one of the following types:

І - Question with answer "Yes" or "No" (2),
ІІ - Open-ended questions. Its answer can contain one or few words (4),
ІІІ - Open-ended questions with many answers. Answers should be divided by newline via pressing "Enter" key (8).
IV - Test question with one correct answer (2).
V - Test questions with multiple correct answers (6).



User can check own results using "User" button in header.

You can use help of the AI (Chat-GPT 4.o) assistant using "AI" button in the header.
